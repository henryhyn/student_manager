student_manager
===============

练习项目: 学生管理系统

在本章的开发中也会采用 MVC 三层网站构架, 它们具体使用的技术如下:

-   M, 模型层, 使用 DAO;
-   V, 视图层, 使用 JSP;
-   C, 控制层, 使用 Servlet.

注意, 以下是我所使用的系统和工具.

-   操作系统: Ubuntu 11.10
-   数据库: MySQL 5
-   JDK 环境: Java 6
-   集成开发环境: IntelliJIDEA 12
-   构建工具: Maven 3
-   Web 容器: Tomcat 7
-   版本控制: Git 1.7
